## Content

autoresearch-for-ols

An experiment using AI-driven iteration to improve an OLS model on the Ames Housing dataset:

- ~100 guided experiments  
- ~25% improvement in RMSE  
- Higher accuracy than a manually built model  

Takeaway: AI improves performance in no time, but simpler models remain easier to interpret and maintain. Choose what works best for you.
